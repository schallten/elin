ELIN

A little bytecode language made to run on System Operating and other tiny, low-power hardware. It's not a fully serious language, just a fun way to mess with stacks and bits.

Read the docs Play with the visualizer

Runs on almost nothing

It fits in about 32KB of RAM. Good for when you don't have much space to work with.

Just a Stack

Everything is a stack. No registers or fancy stuff, just pushing and popping values until it works.

Built for System Operating

Mostly designed for System Operating. If you're using that, this should work fine.

What the code looks like

# Just a simple example
let int radius = 10;
let int pi = 3; 

func int area int r;
    let int res = r * r;
    return res * pi;
end;

print area radius; # Should output 300

Not a big deal

This is just a hobby thing. There's no telemetry or weird stuff because there's no room for it anyway.
